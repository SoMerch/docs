# SoMerch Docs

Public documentation for SoMerch workflows, platform features, fulfillment process, EU shipping, approvals, and managed merch operations.

**SoMerch** is a software-powered corporate merch platform and service partner for distributed companies across Europe.

We help teams plan, produce, store, and ship branded merch from one place.

---

## What you can find here

This documentation explains how SoMerch works across the full merch lifecycle:

- product selection
- quote requests
- order management
- approvals
- production
- warehousing
- inventory visibility
- kitting
- fulfillment
- multi-address shipping
- employee gifting
- platform access
