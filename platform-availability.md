# Platform Availability

SoMerch is available across web, mobile, and desktop platforms.

This helps teams manage merch operations from different environments, whether they are working from a browser, mobile device, or desktop app.

---

## Available platforms

| Platform | Link |
|---|---|
| Web App | [app.somerch.co](https://app.somerch.co) |
| iOS | [Download from the App Store](https://apps.apple.com/us/app/somerch-company-merch/id6771002983) |
| Android | [Get it on Google Play](https://play.google.com/store/apps/details?id=co.somerch.app) |
| Windows | [Get it from Microsoft Store](https://apps.microsoft.com/detail/9nlpq3cps6k5) |

---

## Web app

The SoMerch web app is designed for managing corporate merch workflows from a browser.

Typical web app tasks include:

- browsing the product catalog
- requesting quotes
- reviewing orders
- checking stock
- managing approvals
- tracking fulfillment
- reviewing shipment status

---

## Mobile apps

The iOS and Android apps help teams access SoMerch while on the move.

Typical mobile use cases include:

- checking order status
- reviewing merch options
- following shipment progress
- approving or reviewing requests
- coordinating with internal stakeholders

---

## Windows desktop app

The Windows desktop app gives teams another way to access SoMerch from their working environment.

It is useful for teams that prefer desktop-based workflows for daily operations, procurement, and order tracking.
